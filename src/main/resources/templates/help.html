Common Questions

  • How to create a new project?
  • How to edit an existing project?
  • How to delete a project?

How to create a new project?

To create a new project, click on the "Create New Project" button on the Manage Projects page. Fill in the required details and submit the form.

How to edit an existing project?

To edit an existing project, click on the "Edit" button next to the project you want to modify. Update the project details and submit the changes.

How to delete a project?

To delete a project, click on the "Delete" button next to the project you want to remove. Confirm the deletion to permanently remove the project.
